# Privacy Policy / プライバシーポリシー

**AjoccToys Chrome Extension**

Last updated: 2026-03-09

---

## English

This extension does **not** collect, transmit, or store any personal data.

**What this extension does:**
- Reads the race result page on `data.cyclocross.jp` to convert lap time formats
- Stores only your ON/OFF preference locally on your device using Chrome's built-in `storage.local` API

**What this extension does NOT do:**
- Collect any personal information
- Send any data to external servers
- Track your browsing activity
- Use cookies or fingerprinting

All processing happens entirely within your browser. No data ever leaves your device.

---

## 日本語

本拡張機能は、いかなる個人情報も収集・送信・保存しません。

**本拡張機能が行うこと:**
- `data.cyclocross.jp` 上のレース結果ページを読み取り、ラップタイムの形式を変換する
- Chrome 標準の `storage.local` API を使用して、拡張機能の ON/OFF 設定のみをデバイス上にローカル保存する

**本拡張機能が行わないこと:**
- 個人情報の収集
- 外部サーバーへのデータ送信
- 閲覧履歴のトラッキング
- クッキーやフィンガープリンティングの使用

すべての処理はブラウザ内で完結します。データがデバイス外に送信されることは一切ありません。
