# スクリーンショット画像ガイド

GitHub Pagesのindex.htmlで使用するスクリーンショット画像の要件です。

## 必要な画像一覧

### 1. screenshot-before-after.png
**内容**: 変換前と変換後のテーブル比較

**撮影方法**:
- シクロクロス公式リザルトページ（例: https://data.cyclocross.jp/race/24830）を開く
- ブラウザの拡張機能をOFFにした状態でスクリーンショット（変換前）
- 拡張機能をONにした状態でスクリーンショット（変換後）
- 画像編集ソフトで左右に並べて1枚の画像にする

**表示内容**:
- 左側: 経過時間（19:22.6, 29:20.6など）
- 右側: ネットラップタイム（9:54.5, 9:58.0など）
- 変換後のセルには緑色の背景と⏱アイコンが表示されている状態

**推奨サイズ**: 1200x600px
**ファイルパス**: `docs/images/screenshot-before-after.png`

---

### 2. screenshot-best-lap.png
**内容**: ベストラップのハイライト表示

**撮影方法**:
- シクロクロス公式リザルトページを開く
- 拡張機能をONにして、ベストラップが強調表示された状態でスクリーンショット
- 複数の選手のラップタイムが見えるようにスクロール位置を調整

**表示内容**:
- 個人ベストラップ: 黄色背景 + ★アイコン
- レース全体ベストラップ: オレンジ背景 + 🏆アイコン
- 複数の選手のラップタイムが表示され、それぞれのベストが強調されている

**推奨サイズ**: 1200x600px
**ファイルパス**: `docs/images/screenshot-best-lap.png`

---

### 3. screenshot-graph.png
**内容**: グラフ表示機能の全体像

**撮影方法**:
- シクロクロス公式リザルトページを開く
- 3〜5名の選手を選択してグラフを表示
- グラフ全体が見えるようにスクリーンショット

**表示内容**:
- 折れ線グラフで複数選手のラップタイム推移
- 異なる色で3〜5名の選手が表示されている
- グラフ下部に凡例
- X軸: 周回数、Y軸: タイム（MM:SS形式）

**推奨サイズ**: 1200x700px
**ファイルパス**: `docs/images/screenshot-graph.png`

---

### 4. screenshot-controls.png
**内容**: グラフコントロールUI

**撮影方法**:
- テーブル上部の説明セクションとコントロールエリアが見えるようにスクリーンショット
- 選手名の左側のチェックボックスも含める

**表示内容**:
- テーブル上部の説明セクション（💡 グラフ表示: 選手名の左側のチェックボックスを選択すると...）
- 選手名の左側にチェックボックスが表示されている
- グラフコントロールエリア（1周目トグル、クイック選択ボタン、選手数表示）
- 「上位3名」「全選手」「すべて解除」ボタン

**推奨サイズ**: 1200x500px
**ファイルパス**: `docs/images/screenshot-controls.png`

---

### 5. screenshot-install.png
**内容**: Chrome拡張機能のインストール画面

**撮影方法**:
- Chromeで `chrome://extensions/` を開く
- デベロッパーモードをONにする
- 「パッケージ化されていない拡張機能を読み込む」ボタンが見えるようにスクリーンショット

**表示内容**:
- chrome://extensions/ のページ
- デベロッパーモードがONになっている状態（右上のトグル）
- 「パッケージ化されていない拡張機能を読み込む」ボタンが強調されている（矢印やハイライトを追加すると分かりやすい）

**推奨サイズ**: 1200x600px
**ファイルパス**: `docs/images/screenshot-install.png`

## 画像の準備が完了したら

1. すべての画像を `docs/images/` ディレクトリに配置
2. ファイル名が上記と一致していることを確認
3. `docs/index.html` の該当部分を更新:

```html
<!-- プレースホルダーを実際の画像に置き換え -->
<div class="placeholder-image">...</div>
↓
<img src="images/screenshot-before-after.png" alt="変換前後の比較">
```

## 画像編集のヒント

- **スクリーンショットツール**: macOSなら Cmd+Shift+4、Windowsなら Win+Shift+S
- **高解像度**: Retinaディスプレイで撮影する場合、2倍のサイズになるので縮小してください
- **注釈追加**: 重要な部分に矢印や枠線を追加すると分かりやすくなります
- **ファイル形式**: PNG形式を推奨（透過が不要なら JPEGでも可）
- **ファイルサイズ**: 圧縮ツール（TinyPNG など）で最適化するとページ読み込みが速くなります

## 画像のプレビュー

画像を配置したら、ローカルで `docs/index.html` をブラウザで開いて確認してください。

```bash
# シンプルなHTTPサーバーを起動（Python 3）
cd docs
python3 -m http.server 8000

# ブラウザで http://localhost:8000 を開く
```
